What Makes an App "Compatible" with Intune?

  • ‚úÖ Silent installation supported: Has /quiet, /silent, or /S parameter
  • ‚úÖ Standard exit codes: Returns 0 for success, non-zero for failure
  • ‚úÖ No user interaction required: Fully automated install
  • ‚úÖ Single executable or MSI: One-step installation
  • ‚úÖ Registry-based detection: Can verify install via registry key
  • ‚úÖ File-based detection: Can verify install via file version
  • ‚úÖ Lightweight (< 8GB): Fits within Intune app size limits
  • ‚úÖ No domain dependency: Works on Azure AD-joined devices

üîç How to Identify Incompatible Applications:

Method 1: Automated PowerShell Script

# PowerShell script to analyze ConfigMgr applications
# Run on ConfigMgr site server

Import-Module "$($ENV:SMS_ADMIN_UI_PATH)\..\ConfigurationManager.psd1"
$SiteCode = Get-PSDrive -PSProvider CMSITE
Set-Location "$($SiteCode.Name):\"

# Get all applications
$apps = Get-CMApplication

$incompatibleApps = @()

foreach ($app in $apps) {
    $deploymentTypes = Get-CMDeploymentType -ApplicationName $app.LocalizedDisplayName
    
    foreach ($dt in $deploymentTypes) {
        $flags = @()
        
        # Check for user interaction
        if ($dt.InstallationBehaviorType -eq "InstallForUser") {
            $flags += "User interaction required"
        }
        
        # Check for complex installers
        if ($dt.Technology -eq "Script" -and $dt.InstallCommandLine -notmatch "(/quiet|/silent|/S)") {
            $flags += "No silent install switch"
        }
        
        # Check for Task Sequence dependencies
        if ($dt.Dependencies.Count -gt 5) {
            $flags += "Complex dependency chain"
        }
        
        if ($flags.Count -gt 0) {
            $incompatibleApps += [PSCustomObject]@{
                ApplicationName = $app.LocalizedDisplayName
                DeploymentType = $dt.LocalizedDisplayName
                Issues = $flags -join ", "
                Priority = if ($dt.NumberOfUsersWithRequest -gt 100) { "High" } else { "Medium" }
            }
        }
    }
}

# Export results
$incompatibleApps | Export-Csv -Path "IncompatibleApps_Report.csv" -NoTypeInformation
Write-Host "Found $($incompatibleApps.Count) potentially incompatible applications"
$incompatibleApps | Format-Table -AutoSize
